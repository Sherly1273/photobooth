# Photobooth
Aplikasi photobooth berbasis web dengan HTML, CSS, dan JavaScript.

## Tampilan
![Halaman Utama](photobooth.png)

## Fitur
- Pilih frame foto
- Ambil 3 foto dengan countdown
- Download hasil foto

## Cara Pakai
1. Buka main.html
2. Klik Start
3. Pilih frame
4. Ambil foto
5. Download hasil
